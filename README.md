# PRÁCTICA 9 ANIMACIÓN

Para la práctica 9 he decidido usar uno de los ejemplos básicos dados en la clase práctica, este es el de las físicas con un muro de bloques.
Los cambios que he decidido realizar han sido el añadido de mas muros para formar un bloque mas grande, unos muros al rededor para que los bloques no salgan del area y el cambio de las balas normales por unas explosivas.
El añadido de los muros es útil para seguir probando los explosivos y ver el funcionamiento de las físicas y como los bloques actuan con el mundo y rebotan con las paredes, techo, suelo o entre ellos.

## CONTROLES

Para poder moverse por el mapa bastará con pulsar las teclas "w" para adelante, "s" para atrás, "a" para la izquierda y "d" para la derecha. Para disparar las "granadas" bastará con pulsar el click en el lugar donde se
quiere lanzar. También se puede mover la cámara para mirar a otras direcciones con las flechas del teclado, aunque no recomiendo mirar ni arriba ni abajo ya que puede volverse un poco confuso el movimiento.
